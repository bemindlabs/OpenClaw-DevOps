## User Story

**As a** potential customer,
**I want** to see clear pricing options,
**So that** I can choose the plan that best fits my needs and budget.

## Description

Create a pricing table section with 3 tiers (Free, Pro, Enterprise). Each tier should display price, features included, and a CTA button. Include a toggle for monthly/annual billing with a discount indicator.

## Acceptance Criteria

- [ ] Three pricing tiers displayed: Free, Pro, Enterprise
- [ ] Monthly/Annual toggle with discount shown (e.g., "Save 20%")
- [ ] Each tier shows: name, price, billing period, features list, CTA
- [ ] Popular/recommended tier visually highlighted
- [ ] Feature comparison shows what's included/excluded per tier
- [ ] Enterprise tier has "Contact Sales" CTA instead of price
- [ ] Responsive: stacked on mobile, side-by-side on desktop
- [ ] Accessible: proper table semantics or role attributes

### Pricing Card Component

```tsx
// components/landing/pricing-card.tsx
export function PricingCard({ tier, isAnnual }: Props) {
  const price = isAnnual ? tier.annualPrice : tier.monthlyPrice;

  return (
    <Card className={cn(
      "p-8",
      tier.highlighted && "border-primary ring-2 ring-primary"
    )}>
      {tier.highlighted && (
        <Badge className="mb-4">Most Popular</Badge>
      )}
      <h3 className="text-2xl font-bold">{tier.name}</h3>
      <div className="mt-4 mb-6">
        {price !== null ? (
          <>
            <span className="text-4xl font-bold">${price}</span>
            <span className="text-muted-foreground">/month</span>
          </>
        ) : (
          <span className="text-2xl font-semibold">Contact Sales</span>
        )}
      </div>
      <ul className="space-y-3 mb-8">
        {tier.features.map((feature) => (
          <li key={feature} className="flex items-center gap-2">
            <Check className="h-5 w-5 text-green-500" />
            {feature}
          </li>
        ))}
      </ul>
      <Button className="w-full" variant={tier.highlighted ? "default" : "outline"}>
        {tier.cta}
      </Button>
    </Card>
  );
}
```
