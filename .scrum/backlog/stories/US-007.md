## User Story

**As a** potential customer,
**I want** to see the key features of OpenClaw in an organized grid,
**So that** I can quickly understand the capabilities and benefits of the platform.

## Description

Create a responsive features grid section displaying 6-9 feature cards. Each card should include an icon, title, and brief description. The grid should adapt from 1 column on mobile to 3 columns on desktop.

## Acceptance Criteria

- [ ] Displays 6-9 feature cards in a responsive grid
- [ ] Each card has: icon, title (2-4 words), description (1-2 sentences)
- [ ] Grid layout: 1 col (mobile), 2 col (tablet), 3 col (desktop)
- [ ] Cards have consistent height within each row
- [ ] Hover states for interactive feedback
- [ ] Icons are SVG or icon library (Lucide recommended)
- [ ] Accessible: proper semantic markup, alt text for icons
- [ ] Animation on scroll (optional, respects reduced motion)

### Component Structure

```tsx
// components/landing/features.tsx
interface Feature {
  icon: LucideIcon;
  title: string;
  description: string;
}

const features: Feature[] = [
  {
    icon: Layers,
    title: "Multi-Provider Support",
    description: "Route to OpenAI, Anthropic, Google, and more from a single API."
  },
  // ... more features
];

export function Features() {
  return (
    <section className="py-20">
      <div className="container">
        <h2 className="text-3xl font-bold text-center mb-12">
          Everything you need for AI at scale
        </h2>
        <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
          {features.map((feature) => (
            <FeatureCard key={feature.title} {...feature} />
          ))}
        </div>
      </div>
    </section>
  );
}
```

### Card Component

```tsx
// components/landing/feature-card.tsx
export function FeatureCard({ icon: Icon, title, description }: Feature) {
  return (
    <Card className="p-6 hover:shadow-lg transition-shadow">
      <Icon className="h-10 w-10 text-primary mb-4" aria-hidden="true" />
      <h3 className="text-xl font-semibold mb-2">{title}</h3>
      <p className="text-muted-foreground">{description}</p>
    </Card>
  );
}
```
