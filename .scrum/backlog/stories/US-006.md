## User Story

**As a** visitor to the landing page,
**I want** to see a compelling hero section with clear call-to-action,
**So that** I immediately understand what OpenClaw offers and how to get started.

## Description

Create an impactful hero section for the landing page featuring a headline, subheadline, primary CTA button, and optional visual element (illustration or background gradient). The section should be fully responsive and optimized for conversions.

## Acceptance Criteria

- [ ] Headline clearly communicates the value proposition
- [ ] Subheadline provides supporting context (1-2 sentences)
- [ ] Primary CTA button is prominent and action-oriented
- [ ] Secondary CTA available (e.g., "Learn More")
- [ ] Responsive design: mobile, tablet, desktop
- [ ] Animations are subtle and performant
- [ ] Accessible: proper heading hierarchy, button labels
- [ ] Loads above the fold without layout shift (CLS < 0.1)
- [ ] Dark mode support

### Responsive Breakpoints

| Breakpoint | Headline Size | Layout |

| Mobile (<640px) | 2.25rem (text-4xl) | Stacked |
| Tablet (640-1024px) | 3rem (text-5xl) | Stacked |

### Animation Guidelines

- Use CSS transitions (not JS animations)
- Respect `prefers-reduced-motion`
- Max animation duration: 300ms for UI, 500ms for entrance
